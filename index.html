Over mij

Hallo! Mijn naam is Ivo Jongmans. In december 2018 ben ik begonnen met programmeren. Ik ben gestart met HTML en heb mijzelf gaandeweg verrijkt met CSS, JavaScript, Bootstrap en jQuery. Op dit moment volg ik een opleiding bij CodeGorilla in Groningen om junior webdeveloper te worden en mijn skillset uit te breiden (JSON, AJAX, PHP & SQL). Ik heb de afgelopen maanden veel geleerd en hoop een mooie carrière te kunnen opbouwen met de skills die ik heb toegevoegd aan mijn repertoire. Je kunt mijn progressie via deze site en mijn GitHub bijhouden. Via het formulier onderaan deze pagina kunt u contact met mij opnemen.

HTML

In 2018 ben ik begonnen met het programmeren in HTML. Deze programmeertaal ligt nu bijna altijd ten grondslag aan mijn projecten. Van simpele snippet tot volledige website. HTML is mijn 'gateway' geweest naar het programmeren en zal altijd een vaste waarde zijn binnen mijn skillset.

CSS

Ik gebruik CSS om een groot deel van mijn HTML elementen te stylen en positioneren. Een gridje hier en een flexboxje daar. Met CSS leerde ik voor het eerst repsonsive websites maken door middel van media queries.
